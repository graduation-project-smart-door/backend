# Smart Door Backend

# Установка проекта

- скопировать репозиторий
    ```console
    $ git clone https://github.com/graduation-project-smart-door/backend.git -b backend/production smart-door-backend
    $ cd smart-door-backend
    ```
- установить `.env` файл
- запустить контейнер
    ```console
    $ docker-compose up --build -d
    ```
